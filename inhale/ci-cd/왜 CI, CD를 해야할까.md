# CI
>[!info]
>CI는 코드의 **품질과 통합에 집중**

운영 **배포 전 단계까지 자동화**하는 것이 주요 역할이다.

- 코드 버전 관리: **주기적인 커밋 푸시, 머지**
- 빌드, 테스트 자동화: **빌드와 유닛 테스트 통과를 검증해서 오류 방지**
# CD
>[!info]
> CD는 **배포에 집중**

빌드된 **결과물을 자동으로 운영에 배포하는 것**이 주요 역할이다.

- 컨테이너 배포: 도커 허브 등의 **컨테이너 레지스트리에 푸시**
- 코드 배포: **클라우드나 온프레미스 서버로 배포**
